# Fast Input Flow Optimization

## Objective
Enable input completion in under 10 seconds.

## Workflow
1. **Reduce fields** to minimum.
2. **Use defaults** wherever possible.
3. **Keep flow** in one screen.
4. **Optimize input types** (dropdown, auto-fill).

## Validation
- Minimal inputs.
- Single screen flow.

## Expected outputs
- Optimized UX recommendations.

## Constraints
- Prioritize speed over completeness.
